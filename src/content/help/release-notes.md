Every app update ships with a short release note. To read them:

1. Open {{icon:gear}} **Settings**.
2. Pick the **Release notes** tab.

The newest update is at the top, and the line above the list tells you which version you're on right now. Each note shows:

- The **version** (like v2.944) and the date it went live.
- A badge — {{chip:green|New}} for new features, {{chip:yellow|Fix}} for bug fixes.
- A couple of bullets on what changed and where to find it.

:::example Missing something you saw yesterday?
Check the newest notes first — buttons and tabs sometimes move as part of an update, and the note says where they went.
:::

Older updates collapse behind a **Show earlier updates** button so the list stays quick to scan.

Looking for how to *get* the newest version? See the guide on updating the app when a new version is ready.
